Cosine Similarity Algorithm for Document Comparison

Welcome to my midterm project website. In this project, I will present an algorithm for finding the cosine similarity between 50 documents. This algorithm can be used to compare documents and determine how similar they are to each other.

Algorithm Steps

  1. Convert each word in the documents to its lowercase form.
  2. Remove any non-alphanumeric characters from the documents.
  3. Create a set of all unique words in the documents.
    4. Create a document-term matrix that represents the frequency of each word in each document.
  4. Normalize the columns of the document-term matrix to produce the term frequency-inverse document frequency (TF-IDF) matrix.
  5. Calculate the cosine similarity between each pair of documents using the TF-IDF matrix.
